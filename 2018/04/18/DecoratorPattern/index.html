装饰者模式简要介绍

对象增强的手段

  • 继承
    被增强的对象固定的
    增强的内容也是固定的
  • 装饰者模式
    被增强的对象是可以切换的
    增强的内容是固定的
  • 动态代理（AOP）
    被增强的对象可以切换：Service
    增强的内容也可以切换：事务处理

装饰者模式

  1. 增强的内容是不能修改的！
  2. 被增强的对象可以是任意的！

class 咖啡类 {}

class 有糖咖啡 extends 咖啡类 {
}

class 加奶咖啡 extends 咖啡类 {
}

class 加盐咖啡 extends 咖啡类 {
}

需求：加糖、加奶 的咖啡

咖啡 a = new 加糖();
咖啡 b = new 加盐(a);//对a进行装饰，就是给a加盐
咖啡 c = new 加奶(b);
